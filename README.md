## 📂 Project Highlights

본 포트폴리오에는 다음과 같은 주요 프로젝트들이 포함되어 있습니다.

### 🏢 Corporate Website (기업 홈페이지)

- **Flex Tools**: 제품 일련번호 등록 및 CMS 연동 구축 (PHP, MySQL, 그누보드)
- **V-Golf**: GSAP 기반 스크롤 인터랙션 브랜드 페이지 (Gsap, ScrollTrigger)
- **TXR**: 기업 공시정보 API 연동 및 게시판 관리 시스템

### 📚 E-Catalog (전자 카탈로그)

- **Wisenut & DT**: React와 Framer Motion을 활용한 인터랙티브 전자 카탈로그 구축.
  - 정적인 PDF 카탈로그를 웹 기반의 동적 경험으로 전환
  - 컴포넌트 모듈화 및 반응형 UI 구현

### 🚀 Side Project

- **Pokedex**: PokeAPI를 활용한 포켓몬 도감 웹사이트 (React, Tailwind CSS)
  - 검색, 필터링, 무한 스크롤 및 애니메이션 구현
